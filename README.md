# Тесты для подготовки к экзамену по курсу «Операционные системы»

Этот раздел предназначен для самостоятельной подготовки студентов к экзамену.

Тесты охватывают главы 10–17 основного курса: сетевую среду Linux, разработку в Linux, Git и GitHub, сокеты, асинхронное программирование, безопасность, веб-серверы и развертывание серверного приложения.

Каждый тест содержит 18 вопросов. В каждом вопросе есть 4 варианта ответа и только один правильный ответ.

## Как пользоваться

1. Откройте нужный тест.
2. Выберите один вариант ответа в каждом вопросе.
3. Если ответ правильный, он выделится зеленым.
4. Если ответ неправильный, выбранный вариант выделится красным, а правильный вариант — зеленым.
5. После завершения теста нажмите «Показать результат».
6. Повторите темы, в которых были ошибки.

## Важно для интерактивности

Интерактивная проверка ответов работает на опубликованном сайте GitHub Pages через `index.html` и Docsify.

При обычном просмотре `.md` файла прямо на github.com JavaScript может не выполняться, поэтому студентам лучше давать ссылку на опубликованный GitHub Pages сайт.

## Рекомендуемый порядок подготовки

1. Сначала повторите теоретический материал соответствующей главы.
2. Затем пройдите тест без подсказок.
3. Отметьте темы, в которых были ошибки.
4. Вернитесь к лекции и практическим заданиям по этим темам.
5. Пройдите тест повторно через некоторое время.
